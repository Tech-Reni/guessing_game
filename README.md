# 🎯 Rust Guessing Game

A simple command-line number guessing game built with Rust. This project is based on the classic Rust beginner exercise and demonstrates core concepts like user input handling, random number generation, error handling, and control flow.

---

## 📌 Features

- Generates a random secret number
- Accepts user input from the terminal
- Validates and parses input safely
- Provides feedback (`Too small`, `Too big`, `Correct`)
- Handles invalid input without crashing
- Looping gameplay until the correct guess is made

---

## 🧠 Concepts Covered

- Variables and mutability
- `match` expressions
- Error handling with `Result`
- Looping with `loop`
- User input via `std::io`
- Random number generation using the `rand` crate
